# Price Analysis and Visualization

This project involves web scraping camera accessory prices, storing historical data, and visualizing price trends using Python.

## Overview
The Price Analysis and Visualization project aims to gather price information for camera accessories, store historical price data, and visualize the price trends over time, daily email containing the curent price and graph

## Technologies Used
- Python
- BeautifulSoup
- SQLite
- Matplotlib
- smtplib (for email notifications)
